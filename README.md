# Исследование мобильной зависимости

## Описание темы

Опросник изучает проблему мобильной зависимости среди студентов по трём направлениям.
1. Экранное время. Исследуется количество часов в телефоне в целом, а также в соцсетях, играх и видео. Анализируется частота проверки телефона и привычки использования: после пробуждения, перед сном, во время еды.
2. Отвлечение. Оценивается, как часто телефон отвлекает от учёбы и важных задач, используется ли он на лекциях в неучебных целях. Также изучается влияние телефона на сон и успеваемость.
3. Попытки контроля. Изучаются попытки сократить экранное время, используемые методы (лимиты, таймеры, режим "Не беспокоить"), их успешность. Анализируется реакция при отсутствии телефона и самооценка зависимости.

## Технологии
Python 3.9+
Streamlit
Firebase Firestore
Pandas
Plotly

## Установка и запуск

1. Установите зависимости:
pip install -r requirements.txt
2. Положите файл serviceAccountKey.json в папку с проектом
3. Запустите приложение:
streamlit run app.py

## Структура опроса
35 вопросов
4 блока: основная информация, экранное время, отвлечение, контроль
